Quick Start

1

Install

$ curl -fsSL https://raw.githubusercontent.com/algimantask/rknowledge/main/install.sh | bash
2

Initialize & configure

$ rknowledge init # Start Neo4j + create config
$ rknowledge auth # Configure LLM provider
3

Build your knowledge graph

$ rknowledge build ./documents/ --provider ollama
# âœ… Extracted 40 relations from 4 chunks
# âœ¨ Graph: 54 nodes, 383 edges
4

Query & visualize

$ rknowledge query "machine learning"
$ rknowledge viz # Opens browser visualization

Commands

CommandDescription
rknowledge initInitialize config and start Neo4j via Docker
rknowledge authConfigure LLM provider API keys (interactive)
rknowledge build <path>Process documents and build knowledge graph
rknowledge query <q>Search the graph (natural language or cypher: prefix)
rknowledge exportExport to JSON, CSV, GraphML, or Cypher
rknowledge vizOpen interactive graph visualization in browser
